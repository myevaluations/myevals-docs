## Key Statistics

- **1471** of 2284 tables (64.4%) have a primary key
- **1575** of 2284 tables (69.0%) have at least one index
- **196** indexes are currently disabled
- **1,359** foreign key relationships connect the schema
- **2** tables use monthly partitioning (`Api_Response_Audit`, `Api_Response_Audit_Archive`)
