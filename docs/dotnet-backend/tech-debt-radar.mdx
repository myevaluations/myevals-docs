# Tech Debt Radar

> Bubble chart of all 2,422 .NET source files. **X = migration relevance**, **Y = complexity**, **size = lines of code**. Click any bubble to open that file's documentation.

**Reading the chart:** Top-right = migration priority; Bottom-right = quick wins; Top-left = high complexity / low urgency; Bottom-left = stable, low-risk code.

Filter by layer or minimum migration relevance to focus on specific areas. Security and Evaluations web files tend to dominate the top-right quadrant.
