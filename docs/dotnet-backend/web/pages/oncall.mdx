# OnCall File Reference

**4 files** · **136 lines** · Generated 2/20/2026

On-call scheduling module providing integration points for three third-party scheduling systems (Momentum, QGenda, Tangier) plus a native on-call management page. Pages are thin wrappers that set up user context and embed external scheduling tools.

**Key Workflows**

- Initialize user session context for third-party scheduling system integrations
- Embed external on-call scheduling interfaces (Momentum, QGenda, Tangier) within the MyEvaluations shell
