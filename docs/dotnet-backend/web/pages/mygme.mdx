# MyGME File Reference

**1 files** · **167 lines** · Generated 2/20/2026

Single-file module implementing SSO login integration for MyGME (Graduate Medical Education) systems. Uses RC4 encryption for secure token exchange and handles external authentication flow into the MyEvaluations platform.

**Key Workflows**

- Receive SSO request from MyGME external system
- Decrypt RC4-encrypted credentials and validate user
- Authenticate and redirect user into the MyEvaluations session
