## Quick Tips

- **Can't find your file?** Use the [File Reference Index](./file-index) — searchable table of all 2,422 files.
- **Want to see what else a file touches?** Expand any row in a FileReference component for key methods, stored procedures, and business managers.
- **Changing a business module?** Check the [Dependents Panel](./business/evaluations) on the module's overview page — it shows every web page and scheduler that will be affected.
- **Looking for stored procedures?** See [Stored Procedures](./data-access/stored-procedures).
