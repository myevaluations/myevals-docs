# ICC (In-Training / Clinical Competency Committee)

<!-- Content will be enriched by AI parsing scripts -->

## Overview

The ICC module supports Clinical Competency Committee (CCC) operations, which are required by the ACGME for all residency and fellowship programs. The CCC is responsible for reviewing trainee performance data at least semi-annually and making recommendations about trainee progression, including milestone assessments, remediation plans, and promotion or dismissal decisions.

This module aggregates data from evaluations, procedures, duty hours, and portfolios to present a comprehensive trainee profile to committee members. It manages the committee meeting workflow, milestone level assignments, and formal documentation of CCC decisions.

### Key Responsibilities

- **Committee Management**: Define CCC membership, meeting schedules, and quorum requirements
- **Data Aggregation**: Pull trainee performance data from all relevant modules for committee review
- **Milestone Assessment**: Facilitate semi-annual milestone level assignments by CCC members
- **Meeting Workflow**: Structured meeting process with resident review, discussion, and voting
- **Decision Documentation**: Record CCC recommendations (advancement, remediation, probation, dismissal)
- **ACGME Reporting**: Generate milestone data for ACGME WebADS submission

## Business Workflows

### Semi-Annual CCC Review

```mermaid
sequenceDiagram
    participant Chair as CCC Chair
    participant ICCM as ICCManager
    participant IDA as ICCDataAggregator
    participant IRM as ICCReviewManager
    participant IMM as ICCMilestoneManager
    participant DB as SQL Server

    Note over Chair,DB: Phase 1 - Meeting Preparation
    Chair->>ICCM: PrepareMeeting(programId, meetingDate)
    ICCM->>DB: sp_ICC_GetTraineesForReview
    DB-->>ICCM: Trainees due for review

    loop For each trainee
        ICCM->>IDA: AggregateTraineeData(traineeId, reviewPeriod)
        IDA->>DB: sp_Evaluation_GetSummary (evaluations)
        IDA->>DB: sp_Procedure_GetSummary (procedures)
        IDA->>DB: sp_DutyHours_GetCompliance (duty hours)
        IDA->>DB: sp_Portfolio_GetHighlights (portfolio)
        IDA-->>ICCM: ICCTraineeSummaryInfo
    end

    ICCM->>DB: sp_ICC_InsertMeeting
    ICCM->>DB: sp_ICC_InsertReviewItems

    Note over Chair,DB: Phase 2 - Committee Meeting
    Chair->>ICCM: StartMeeting(meetingId)
    ICCM->>DB: sp_ICC_RecordAttendance(members[])

    loop For each trainee
        Chair->>IRM: BeginTraineeReview(traineeId)
        IRM->>DB: sp_ICC_GetTraineeSummary
        DB-->>IRM: Aggregated trainee data

        Note right of IRM: Committee discusses<br/>trainee performance

        Chair->>IMM: AssignMilestoneLevels(traineeId, milestones[])
        IMM->>DB: sp_ICC_InsertMilestoneAssessments

        Chair->>IRM: RecordDecision(traineeId, decision)
        IRM->>DB: sp_ICC_InsertDecision
    end

    Chair->>ICCM: CloseMeeting(meetingId)
    ICCM->>DB: sp_ICC_UpdateMeetingStatus("Complete")
```

### Milestone Level Assignment

```mermaid
sequenceDiagram
    participant CCC as CCC Members
    participant IMM as ICCMilestoneManager
    participant DB as SQL Server

    CCC->>IMM: GetMilestoneFramework(programId)
    IMM->>DB: sp_ICC_GetMilestoneDefinitions
    DB-->>IMM: Competencies + sub-competencies + level descriptors

    CCC->>IMM: GetPreviousAssessments(traineeId)
    IMM->>DB: sp_ICC_GetMilestoneHistory
    DB-->>IMM: Previous milestone levels over time

    CCC->>IMM: AssignLevels(traineeId, assessments[])
    Note right of IMM: Each assessment includes:<br/>- Sub-competency<br/>- Level (1-5)<br/>- Supporting evidence<br/>- Committee notes

    IMM->>DB: sp_ICC_InsertMilestoneAssessments
    IMM->>IMM: CalculateTrajectory(history + current)
    IMM->>DB: sp_ICC_UpdateMilestoneTrajectory
```

### ACGME WebADS Export

```mermaid
sequenceDiagram
    participant PD as Program Director
    participant IRM as ICCReportManager
    participant DB as SQL Server

    PD->>IRM: GenerateWebADSExport(programId, reportingPeriod)
    IRM->>DB: sp_ICC_GetMilestonesByTrainee

    IRM->>IRM: TransformToWebADSFormat()
    IRM->>IRM: ValidateRequiredFields()
    IRM->>IRM: GenerateCSV(webADSSchema)

    IRM-->>PD: WebADS milestone CSV file
    Note right of PD: Upload to ACGME<br/>WebADS portal
```
